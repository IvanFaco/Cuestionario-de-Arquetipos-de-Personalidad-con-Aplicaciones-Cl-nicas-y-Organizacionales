# Cuestionario de Arquetipos de Personalidad

Aplicación interactiva para evaluar arquetipos de personalidad y estructura psicológica Junguiana, con análisis detallado y exportación de reportes.

## 🎯 Características

- **Cuestionario Completo**: 60 preguntas distribuidas en 12 arquetipos + 4 dimensiones estructurales
- **Evaluación en Tiempo Real**: Indicadores visuales de progreso por sección
- **Análisis Detallado**: Diagnóstico personalizado basado en resultados
- **Exportación de Reportes**: 
  - Excel con datos crudos y análisis
  - PDF con gráficos visuales e información detallada
- **Interfaz Intuitiva**: Diseño responsive con colores y elementos visuales atractivos

## 🛠️ Requisitos

- Python 3.8+
- Ver `requirements.txt` para dependencias

## 📦 Instalación

```bash
pip install -r requirements.txt
```

## ▶️ Ejecutar la Aplicación

```bash
streamlit run app.py
```

La aplicación se abrirá en `http://localhost:8501`

## 🚀 Despliegue en Streamlit Cloud

1. Sube este repositorio a GitHub
2. Ve a [Streamlit Community Cloud](https://streamlit.io/cloud)
3. Conecta tu repositorio de GitHub
4. Selecciona la rama `main` y el archivo `app.py`
5. ¡Despliegue completado!

## 📝 Uso

1. Responde las preguntas de manera honesta
2. Los indicadores de sección muestran tu progreso (rojo → verde)
3. Presiona "Generar Análisis Completo" cuando termines
4. Descarga el reporte en Excel o PDF
5. Usa "Reiniciar Test" para hacer otra evaluación

## ⚖️ Notas Importantes

Este instrumento sigue principios de análisis de arquetipos Junguianos. Los resultados son tendencias narrativas, no diagnósticos psiquiátricos concluyentes. Para evaluaciones clínicas profundas, consult a un profesional de salud mental.
